# 🌱 Crop Recommendation System

This ML project recommends crops based on soil and environmental conditions.

## 📊 Features
- Trained multiple models: Random Forest, SVM, KNN, etc.
- Compared using accuracy, confusion matrix, classification report.
- Best model: Random Forest (99.1% accuracy)

## 📁 Dataset
- `Crop_recommendation.csv` from Kaggle
